# Portfolio Website

这是一个基于 Figma 设计完成的个人作品集网站。

## 文件结构

```
web/
├── index.html          # 主 HTML 文件
├── styles.css          # 样式文件
├── script.js           # JavaScript 交互功能
├── images/             # 图片目录
│   └── README.md       # 图片说明
└── README.md           # 本文件
```

## 功能特性

- ✅ 响应式导航栏，带平滑滚动
- ✅ 英雄区域（Hero Section）
- ✅ 技能展示区域（4个技能卡片）
- ✅ 关于我区域
- ✅ 作品集展示（3个项目卡片）
- ✅ 客户评价区域（3个评价卡片）
- ✅ 联系表单
- ✅ 页脚

## 使用方法

1. 确保所有文件都在 `web/` 目录下
2. 下载 Figma 设计中的图片并放入 `images/` 目录：
   - hero-image.png
   - about-image.png
   - project-ahuse.png
   - project-dashboard.png
   - project-easyrent.png
   - avatar-dianne.png
   - avatar-kristin.png
   - avatar-kathryn.png
3. 在浏览器中打开 `index.html` 文件

## 设计规范

### 颜色
- Primary: #5E3BEE
- Heading: #282938
- Body: #1C1E53
- Background Shade: #F5FCFF
- Contact Background: #ECFFFD
- Dribbble: #E62872

### 字体
- 字体族: Roboto
- 字体大小: 18.67px - 74.67px（根据元素不同）
- 字重: 400 (Regular), 600 (Semi-bold), 700 (Bold)

### 间距
- 容器最大宽度: 1749.33px
- 内边距: 85.33px - 149.33px（根据区域不同）
- 元素间距: 10.67px - 114.67px（根据元素不同）

## 浏览器支持

- Chrome (最新版本)
- Firefox (最新版本)
- Safari (最新版本)
- Edge (最新版本)

## 注意事项

- 图片需要从 Figma 设计中导出
- 所有尺寸和间距都严格按照 Figma 设计规范
- 网站已包含响应式设计，适配不同屏幕尺寸
