## Note:
The images should be downloaded from the Figma file. You can use placeholder images for testing purposes, but for the final version, please download the actual images from the Figma design.

To download images from Figma:
1. Open the Figma file
2. Select each image element
3. Export as PNG (2x scale for better quality)
4. Save them in this directory with the names listed above
